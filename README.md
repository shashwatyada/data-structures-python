# 🚀 Data Structures & Algorithms in Python

Welcome to my DSA repository! This repository serves as a practical playground where I implement core Data Structures and solve classic interview problems from scratch using **Python**.

---

## 🛠️ Tech Stack & Tools
* **Language:** Python 3.14+
* **IDE:** Visual Studio Code (VS Code)
* **Version Control:** Git & GitHub (Managed via command line)

---

## 📂 Repository Structure & Topics Covered

### 1. 🧵 Strings & Arrays (`array_string.py`)
Basic operations practicing dynamic array behavior and string manipulation:
* **String Operations:** Indexing, slicing, in-place reversal, and string splitting.
* **Array (List) Operations:** Dynamic insertions (`append`, `insert`), deletions (`pop`, `remove`), slicing, and linear traversal.

### 2. 🔗 Singly Linked List (`linked_list.py` & `linked_list_next.py`)
Implemented a dynamic singly linked list from scratch, including memory allocation concept via custom `Node` class.
* **Core Operations:** Insert at beginning, insert at end, delete node, and sequential traversal/display.
* **LeetCode / Interview Solutions implemented:**
  * **Reverse a Linked List** (In-place reversal using 3-pointer iterative approach) - *Time: $O(N)$ | Space: $O(1)$*
  * **Find Middle of Linked List** (Optimized using Tortoise & Hare / Fast-and-Slow pointers) - *Time: $O(N)$ | Space: $O(1)$*
  * **Linked List Cycle Detection (LeetCode 141)** (Detecting loops using pointer collision) - *Time: $O(N)$ | Space: $O(1)$*
